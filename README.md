# НейроИсследователь

Образовательная веб-платформа для детей **10–14 лет** (РФ): осознанное знакомство с нейросетями — от простых идей до мини-проектов. Мобильные приложения — после стабильного веб-MVP.

## Flutter-приложение (скелет)

В репозитории есть базовый клиент на Flutter: приветствие → дерево уровней (заглушка).

```bash
cd НейроИследователь
flutter pub get
flutter run -d chrome   # веб
# или: flutter run       # устройство / эмулятор
```
