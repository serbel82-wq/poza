# Звуковые эффекты для Neuro Explorer

## Добавление звуков

Добавьте MP3 файлы в эту папку:
- `success.mp3` - Звук успешного завершения
- `xp.mp3` - Звук получения XP
- `complete.mp3` - Звук завершения урока
- `click.mp3` - Звук нажатия
