# Анимации для Neuro Explorer

## Использование Lottie анимаций

Добавьте JSON файлы анимаций в эту папку и используйте так:

```dart
import 'package:lottie/lottie.dart';

Lottie.asset('assets/animations/success.json')
```

## Доступные анимации

- `success.json` - Анимация успешного завершения
- `xp.json` - Анимация получения XP
- `loading.json` - Анимация загрузки
- `celebration.json` - Праздничная анимация
- `robot.json` - Анимация робота-ассистента
- `star.json` - Анимация звезды

## Где брать анимации

- https://lottiefiles.com/ - бесплатные Lottie анимации
- https://rive.app/ - интерактивные анимации
