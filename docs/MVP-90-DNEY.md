# MVP «НейроИсследователь» (~90 дней, соло, веб)

## Цель MVP

Доказать гипотезу: **семья готова платить**, если ребёнок **регулярно** (не разово) проходит структурированный курс с понятным прогрессом и **итоговым мини-проектом**, а родитель видит прогресс и темы.

## Целевая аудитория

- Ребёнок **10–14 лет**, базовая грамотность в браузере.
- Родитель оплачивает; нужен **прозрачный** триал и отмена.

## В MVP входит

### Продукт

1. **Регистрация** — email родителя + создание профиля ребёнка (ник, возраст в диапазоне).
2. **Онбординг** — короткое приветствие, правила безопасности (простым языком), согласие родителя с политикой (чекбокс + ссылка на документы-заглушки для юриста).
3. **Сезон 1** — 8 уроков + финальный проект (см. `SEZON-1-KARTA.md`).
4. **Урок** — теория (текст + 1–2 иллюстрации/схемы) + **практика** (задание с отправкой ответа; где возможно — чеклист/автопроверка формата).
5. **Прогресс** — галочки по урокам, процент сезона, дата последней активности (для родителя).
6. **Родительский кабинет** — список тем, прогресс, управление подпиской (отмена).
7. **Подписка в рублях** — триал (например 7 дней) + рекуррент; интеграция одного платёжного провайдера (выбор при реализации).
8. **ИИ в заданиях** — вызовы **только через бэкенд** (ключи не в браузере), лимиты на пользователя/день.

### Технически (минимум)

- **Фронт**: один адаптивный веб-клиент (десктоп приоритет, телефон «читаемо»).
- **Бэкенд**: API, БД (пользователи, уроки, прогресс, платежи webhook).
- **Авторизация**: сессии или JWT; пароли — хэш.
- **HTTPS**, базовая защита от спама регистраций.

## В MVP не входит (отложить)

- Нативные iOS/Android (после метрик веба).
- Соцсеть, лента друзей, чаты между детьми.
- Полноценная песочница кода с произвольным исполнением (только если успеваешь — упростить до «вставь ответ» / внешний embed позже).
- Сложная геймификация (кланы, рейтинги) — только простые бейджи за сезон, если останется время.
- Много языков и курсов — только Сезон 1.

## Критерий «MVP готов к платным тестам»

- 20–50 знакомых/честных тестеров могут пройти сезон без ручной поддержки «на каждый клик».
- Платёж проходит, триал отключается, отписка работает.
- Есть контакт поддержки (email/Telegram).

## Риски соло (смягчение)

| Риск | Что делать |
|------|------------|
| Нет времени на контент | Писать уроки **до** «красивого дизайна»; один сезон — фиксированный объём |
| Юридические документы | Заглушки + **оплачиваемая** правка юриста до массовой рекламы |
| Выгорание | Недельные спринты из `MVP-90-DNEY`; один артефакт в неделю в прод |

## Порядок работ (скелет)

1. Недели 1–2: тексты Сезона 1 + макеты экранов (Figma или бумага).
2. Недели 3–6: бэкенд + auth + модель данных + один урок end-to-end.
3. Недели 7–10: все уроки + родитель + прогресс.
4. Недели 11–12: платежи + полировка + тест с семьями.

Точные сроки подстраиваются под твою занятость; важно **не расширять** объём MVP.
